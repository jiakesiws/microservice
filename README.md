## 其他知识

**ACID**分别是什么？

- A 代表 Aotmicity（原子性）
- C 代表 Consistency （一致性）
- I 代表 Isolation （独立性）
- D 代表 Durability (持久性)

**CAP**又分别是什么？

- C：Consistency （强一致性）
- A：Available （可用性）
- P：Partition tolerance （分区容错性）
